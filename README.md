angular_symfony
===============

There are following package:

 * symfony:
  * FosRestBundle
  * JmsSerializerBundle
 * Angular
  * Bower



in main directory run install composer, update pareter yml and then run run.php

                php run.php

next in main directory run:

                php -S localhost:8080

Go in browser to

               localhost:8080/app_dev.php/homepage
